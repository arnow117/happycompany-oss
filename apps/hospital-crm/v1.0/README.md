# Hospital CRM

医疗器械 CRM 管理系统 -- 杭州示例医疗器械专属。管理医院档案、装机设备跟踪、维保合同、中标信息查询、采购意向监控和客户联系人。

## 功能

- **医院管理**: 多维度医院信息（省/市/区县/等级/床位/收入/渠道），支持 Excel 批量导入
- **设备管理**: 装机设备全生命周期跟踪，按品牌/类别/医院筛选，staging 安全导入
- **维保管理**: 维保合同跟踪，保养进度记录，到期提醒
- **中标信息**: 政府采购中标记录，供应商分析，金额统计
- **采购意向**: 政府采购意向公示，预算监控
- **联系人管理**: 客户关键人信息，时间线备注
- **聚合查询**: 医院全景视图、供应商中标分析、维保到期预警
- **全局搜索**: 跨表关键词搜索（医院/设备/维保/中标/联系人）
- **AI 表格同步**: CRM 数据同步到钉钉 AI 表格
