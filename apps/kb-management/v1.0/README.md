# kb-management

基于文件的知识库管理系统。利用 Claude 的文件读写能力，实现文档入库、结构化存储和语义检索。

## 功能

- **文档入库**: 将文本、文件内容整理为知识卡片，结构化存储
- **知识检索**: 基于关键词和索引的文件检索，综合多源回答
- **知识管理**: 列出、查看、删除知识卡片

## 使用方式

向 bot 发送以下格式的消息：
- "入库：<内容>" — 将内容存入知识库
- "查询：<问题>" — 从知识库中搜索并回答
- "列出知识" — 查看所有知识卡片
- "删除知识：<标题>" — 删除指定卡片
